# Planilha_MEI_Anual_financeiro
Planilha MEI Anual — controle financeiro completo no celular e pc, 12 abas mensais, dashboard automático, controle do limite MEI e DAS. Você lança, a planilha calcula tudo.

INSTRUÇÂO: UTILIZE ESTA PLANILHA COM O Google Planilhas.

Para quem é essa planilha?

Para o MEI que quer saber quanto entra, quanto sai e quanto sobra — sem complicar. Não precisa saber nada de Excel. Não precisa de contador para o controle básico. Você abre no celular, lança seus valores e a planilha faz o resto.

✓ Planilha com 12 abas — uma por mês (até 200 lançamentos por mês)
✓ Aba de resumo anual com totais automáticos
✓ Dashboard com lucro, receitas e % do limite usado
✓ Controle do DAS com cálculo automático de 6%
✓ Alerta quando faturamento passar de 90% do limite
✓ Funciona no Google Sheets — 100% gratuito para usar
✓ Instruções de uso dentro da própria planilha
✓ Limite MEI configurável (acompanha qualquer atualização)
✓ Acesso vitalício — comprou, é seu para sempre
